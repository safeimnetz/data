Allgemeines

Beinahe alle Accounts werden mittels Passwörtern verschlossen, können mit diesen aber auch wieder aufgeschlossen werden. Genau aus diesem Grund ist es wichtig, dass man starke Passwörter verwendet und gut auf diese achtet.

Starke Passwörter erstellen

Beim Erstellen eines guten Passworts sollten Sie die folgenden bewährten Praktiken beachten, um die Sicherheit Ihres Online-Kontos zu erhöhen:

  • Länge : Verwenden Sie lange Passwörter. Ein Passwort mit mindestens 12 Zeichen ist empfehlenswert.
  • Komplexität : Ihr Passwort sollte aus einer Mischung von Großbuchstaben, Kleinbuchstaben, Zahlen und Sonderzeichen bestehen.
  • Einzigartigkeit : Vermeiden Sie die Verwendung von leicht erratbaren Passwörtern wie "Passwort123" oder "123456". Jedes Ihrer Konten sollte ein eigenes, einzigartiges Passwort haben.
  • Keine persönlichen Informationen : Verwenden Sie keine leicht verfügbaren persönlichen Informationen wie Geburtsdaten, Namen von Familienmitgliedern oder bekannte Wörter.
  • Regelmäßige Änderungen : Ändern Sie Ihre Passwörter in regelmäßigen Abständen. Verwenden Sie nicht dasselbe Passwort über einen längeren Zeitraum.
  • Widerstandsfähig gegen Wörterbuchangriffe : Ihr Passwort sollte so gestaltet sein, dass es gegen automatisierte Wörterbuchangriffe und Brute-Force-Angriffe bestehen kann.
  • Regelmäßige Überprüfung : Überprüfen Sie regelmäßig Ihre Konten auf verdächtige Aktivitäten und ändern Sie Ihr Passwort, wenn Sie Anzeichen für eine Kompromittierung feststellen.

Passwort Manager

Daher es schwierig sein kann, sich viele verschiedene Passwörter zu merken, sollte man einen sogenannten “Passwort Manager” verwenden. Dies ist ein Programm, welches alle Passwörter speichert und mithilfe eines “Master Passworts” abrufbar macht. Somit muss man sich nur ein einziges Passwort merken und jeder Account ist mit seinem eigenen, sicheren Passwort geschützt.

Ein Beispiel für einen Passwort Manager ist das kostenlose Tool “ Bitwarden ”, welches es für alle bekannten Plattformen (Windows, MacOS, Android, iOS) gibt. Neben der Möglichkeit seine Passwörter zu speichern, kommt Bitwarden auch mit einem integrierten Passwort Generator.

Ein Problem mit Passwort Managern ist jedoch die Methode, wie die Passwörter gespeichert werden. Sollten diese lokal gespeichert werden, sind sie bei Verlust des Gerätes ebenfalls verschwunden. Bei einer Cloud-Lösung besteht ein höheres Risiko, dass die Daten gestohlen werden. 

Zwei-Faktor-Authentifizierung (2FA)

Zweifaktor-Authentifizierung (2FA), auch als Zwei-Schritt-Verifizierung bezeichnet, ist eine wesentliche Sicherheitsmaßnahme im digitalen Zeitalter.

Hierbei muss auf eine weitere Weise bestätigt werden, dass es sich beim Anmelden wirklich um die besagte Person kümmert. Dies wird meist mittels E-Mail, SMS oder eigene Apps durchgeführt. Die meistens Plattformen unterstützen 2FA und es ist äußerst empfehlenswert, dies überall zu aktivieren.
